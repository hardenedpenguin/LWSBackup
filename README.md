# LWSBackup
Easy backup and restore script that backups esential files on hamvoip or any linux system. Mainly made for a GMRSHub node in GMRS but it could be used in many other systems.

reusable utility for:

HamVoIP, 
, AllStar nodes
, Debian repeaters
, Raspberry Pis
, mini servers
, general homelab backups

especially with:

, menu UI
, restore mode
, cron integration
FTP optional
, dependency auto-install
, config persistence
